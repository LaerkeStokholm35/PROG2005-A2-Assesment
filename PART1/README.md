Single-page TypeScript application for managing inventory items.

## Features
- Add, update, delete inventory items
- Search by item name
- Display all items or filter by popular items
- Input validation and error handling
- Responsive design for mobile devices

## Setup
1. Open `index.html` in a web browser
2. No build process required — runs directly in browser

## Files
- index.html: Main HTML structure
- app.ts: TypeScript logic (compile to app.js)
- style.css: Responsive styling
