This application is a website for managing and keeping track of habits, creating streaks when habits are being done - and keeping track kind of like an online check list for your life.

## Features (v1.0.0)
- Display habit list with categories and descriptions
- Calculate and display current streaks for each habit
- Professional, responsive design
- Clean, minimal UI

## Files
- **index.html**: Main HTML structure and container
- **app.ts**: TypeScript logic (compiles to app.js)
- **app.js**: Compiled JavaScript
- **style.css**: Styling
- **tsconfig.json**: TypeScript compiler configuration
- **package.json**: Project metadata

## Future Features
- Mark habits as completed on specific dates
- View completion history
- Delete/edit habits
